其中V由多个头拼接而成，对于每个头，输出的矩阵的秩受到矩阵尺寸的制约，只能有d_head维。在讲DeepSeek的创新时我们提到，各个head的向量是高度相关的，因此实际上仍然分布在一个低秩空间中，会导致V可能丢失了高维数据分布中一些维度的信息。而Softmax之后的Attention层以及输出层都是线性的，V如果没能包含某些维度的信息，就无法通过用非线性操作去“拟合”的方式来恢复这些信息。

2.不合理的权重分配

以前的许多模型在实际运行时，对于所有token的key都和该query相关度都很低的情形，经常会把大量的注意力分数分配给句子的第一个token（比如起始符），即使这个Token没有任何实际语义。这可能是因为所有token的注意力权重之和强制为1，为了保持数值稳定强行把第一个token作为“垃圾桶”。

标准 Attention （加权平均）： 由于 Softmax 的性质 （所有权重之和为 1 且非负 ), 标准 Attention 的输出本质上是 Value 向量的凸组合 (Convex Combination)0 刀何限制： 输出向量必须落在所有 value 向量构成的凸包 (Convex HuII) 内部。 模型无法生成一个位于这些向量 " 外部 " 或 " 反方向 “ 的向量。 Gated Attention （非线性缩放）： 门控 G e （0， 1） 对向量的每一个维度进行了独立的缩放。 刀何突破： 这相当于对凸包内的点进行了非均匀的拉伸或压缩。 原本必须落在凸包内的占现在可以被 Gate " 推 " 到原点附近 （当 G + O), 或者在某些维度保持原样， 在另一些维度被抑制。 这种操作破坏了 “ 凸组合 " 的几何限制， 使得输出空间变得非凸且极其丰富

实验证明，门控矩阵往往是一个稀疏矩阵，这也解决了上面所述不合理的权重分配问题。
